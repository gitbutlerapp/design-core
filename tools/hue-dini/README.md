# Hue-dini (Svelte 5)

A very small reverse-engineered color ramp playground inspired by grayscale.design.

## Run

```bash
pnpm install
pnpm dev
```

## Features

- 12 editable luminance stops
- Live multi-color swatches generated from luminance
- URL sync via `?lums=`
- Copyable `lums` parameter for reuse
